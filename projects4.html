퇴직연금 전산 아키텍처 설계 및 분석

현대차증권 전사 차세대 시스템 구축 (연금 파트)

  • 프로젝트 성격 : 증권사 퇴직연금 비즈니스 고도화를 위한 차세대 Core 시스템 AS-IS 분석 및 TO-BE 설계
  • 핵심 역할 : 비즈니스 요건 정의, 데이터 모델링 기반 기획, 품질 검증 시나리오 설계

주요 수행 업무

  • AS-IS 분석 및 TO-BE 전략 수립 : 기존 노후화된 연금 시스템의 비즈니스 로직을 전수 조사하고, 차세대 아키텍처에 적합한 기능별 프로세스 재설계
  • 복잡 도메인의 시스템 요건 정의 : 퇴직연금 원장 처리 및 상품 관리 등 증권 도메인 특유의 복잡한 비즈니스 규칙을 분석하여, 차세대 시스템에 최적화된 기능 상세 요건 및 데이터 매핑 정의
  • 품질 보증(QA) 설계 : 시스템 전환 시 비즈니스 결함을 사전에 차단하기 위한 AS-IS 대비 데이터 정합성 검증 시나리오 설계 및 실무 중심의 사용자 수용 테스트(UAT) 케이스 수립

현업 전문성 기반의 전산 기획

  • 데이터 중심의 요구사항 구체화 : 개발자의 시각에서 데이터 흐름(Data Flow)을 이해하고, 기획 단계에서 발생할 수 있는 데이터 정합성 오류를 사전 방지
  • 도메인 가교 역할 : 증권업 특유의 복잡한 금융 상품 구조를 전산 언어로 번역하여, 현업의 요구사항과 개발 구현 가능성 사이의 간극 해소
